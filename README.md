# 设计模式学习

## 一、创建型模式
### 1.单例模式的特点
* 　a、单例类只能有一个实例。
* 　b、单例类必须自己创建自己的唯一实例。
* 　c、单例类必须给所有其他对象提供这一实例。
* 单例模式保证了全局对象的唯一性，比如系统启动读取配置文件就需要单例保证配置的一致性。
